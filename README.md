# The Orchid

## Installation
1. Search for [The Orchid]() in the Visual Studio Code's Extension Marketplace
2. Click **Install**
3. Open the **Command Palette** with <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>P</kbd> or <kbd>⇧</kbd> + <kbd>⌘</kbd> + <kbd>P</kbd>
4. Enjoy
